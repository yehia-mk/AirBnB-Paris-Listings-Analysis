# AirBnB Paris Listings Analysis

Analysis of AirBnB listings in Paris to determine the impact of recent regulations.

## Overview

This project explores AirBnB listing data in Paris, focusing on:

- Average listing prices across neighbourhoods
- Price variations by accommodation capacity in the most expensive neighbourhood
- Trends in new host registrations and pricing over time
- Impact of 2015 regulations on the AirBnB market

## Key Findings

- The number of new AirBnB hosts declined sharply after the 2015 regulations, and listing prices increased as a result.
- Neighbourhood-level price analysis reveals significant variation across Paris.
